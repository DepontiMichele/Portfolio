# SimCity_java
# SimCity Entity Modeling (Java)

### Description
This project was developed as part of my technical training with the goal of applying the principles of **Object-Oriented Programming (OOP)**. 
It is not a complete game engine, but rather a **base structure (Entity Modeling)** designed to define the core classes and their relationships, such as buildings, resources, and public infrastructure management.

### Technologies & Tools
* **Language:** Java
* **IDE:** Apache NetBeans

### Key Implementations:
* **Requirement Analysis:** Defining fundamental entities (e.g., `Building`, `Citizen`, `PublicService`).
* **OOP Principles:** Implementation of inheritance, encapsulation, and polymorphism to model hierarchies between different structure types.
* **Class Relationships:** Defining logical links between entities to simulate system interaction.

### Objective
The focus of this exercise was software architecture: organizing code to be modular, maintainable, and extensible, by applying the best practices learned during the Object-Oriented Programming module.
