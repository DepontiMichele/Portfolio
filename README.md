# Weather Fetcher API
This project is a Python-based utility to fetch real-time weather data using the OpenWeatherMap REST API.

## Features
- Real-time weather data extraction (Temperature, Description).
- Robust error handling for invalid API keys or connection issues.
- Docker-ready for deployment.

## How to run
1. Ensure you have the `requests` library installed: `pip install requests`
2. Add your OpenWeatherMap API Key in `API_CALL.py`.
3. Run the script: `python3 API_CALL.py`

## Technologies
- Python 3.x
- Requests Library
- Docker
